# AltSchool Engineering Profile Page

A simple HTML web page showcasing my profile as an AltSchool Engineering student, including my biography, reasons for joining the program, and my goals.

---

## 📌 Project Description

This project is a straightforward, HTML-only personal profile page designed to display essential information about my journey as an AltSchool Engineering student. The page includes:

- Basic information with ALT SCHOOL ID  
- Biography section  
- Reasons for joining AltSchool  
- Goals for the Engineering program  
- Fragment identifier navigation (Back to Top link)

The project emphasizes simplicity, accessibility, and adherence to HTML standards, as required by the program.

---

## 🛠 Features

- Display of personal and program information  
- Fragment identifier navigation for easy scrolling  
- Strictly HTML-only (no CSS)  
- Clean and organized file structure

## 🚀 How to View the Project

### Option 1: Local Viewing

1. Clone or download this repository  
2. Open `index.html` in any web browser  

### Option 2: Live Hosting

**GitHub Pages (Recommended)**

1. Create a GitHub account if you don't have one  
2. Create a new repository named `alt-school-profile`  
3. Upload `index.html` and `README.md`  
4. Go to **Settings > Pages**  
5. Select **main branch** and **/ (root) folder**  
6. Click **Save** – your site will be live at:  
   `https://<your-username>.github.io/alt-school-profile/`  

**Netlify (Alternative Option)**

1. Create a Netlify account (free tier)  
2. Drag and drop the project folder to the dashboard  
3. Site is deployed instantly with a `.netlify.app` URL
