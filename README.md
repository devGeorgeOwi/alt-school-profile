# AltSchool Engineering Profile Page

A simple HTML web page displaying my basic information as an AltSchool Engineering student, including my biography, reasons for joining the program, and my goals.

## Features

- Basic information section with ALT SCHOOL ID
- Biography section
- Reasons for joining AltSchool
- Goals for the Engineering program

- Strictly HTML-only (no CSS as per requirements)

## How to View the Project

### Option 1: Local Viewing
1. Clone or download this repository
2. Open the `index.html` file in any web browser

### Option 2: Live Hosting (Free Options)

#### GitHub Pages (Simplest Option)
1. Create a GitHub account if you don't have one
2. Create a new repository named `alt-school-profile`
3. Upload your `index.html` and `README.md` files
4. Go to Settings > Pages
5. Select "main" branch and "/ (root)" folder
6. Click "Save" - your site will be live at `https://<your-username>.github.io/alt-school-profile/`

#### Netlify (Alternative Option)
1. Create a Netlify account (free tier available)
2. Drag and drop your project folder to Netlify's dashboard
3. Your site will be deployed instantly with a `.netlify.app` URL
